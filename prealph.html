Current Development Stage

BaeDray Placeholder spritesheet partitioned for rigging

My main goal for this game is create a 2.5D Brawler/Run and Gun. Four years ago this project was entitled Bug Catcher Elliot. This was before I knew anything about Unity. From then I have created and had to recreate the current build for projectLit. During that time period I have familiarized myself with the Unity Framework and Adobe Services, but know I have a lot to learn and utilize.

I have created all game objects sprites myself but these are placeholders at the momement. Currently, I am attempting to master the manipulation of computer graphics. I learned to manipulate Unity's 2D physics system by playing around with attributes and methods attatched to the Rigidbody2D object type to simulate my representation of telekinesis and a few funtions of other superpowers. The current build is only available for PC and Mac.

The W , A, S, and D keys control the characters actions such as walking, jumping. Fighting is controlled by E key as combos and directional keys. Blocking is controlled by Q key. Blocking and Fighting have been turned off currently. The mouse controls the super powers and are represented as a character specific cursor. I coined these super powers as Spirit for this game. A player has health, stamina, and spirit bars that limit character actions. I am currently working on syncronizing player cursor interactions across the Photon Unity Network.
